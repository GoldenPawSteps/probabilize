# Probabilize

Probabilize is a dark-mode, mobile-responsive probability market web app implementing:
- Register (`B = 1`)
- Make market (LMSR with `L = -b ln(min(p))`)
- Take market (`Δq'`, `ΔC` validation)
- Unmake market with gradient-based settlement

## Run

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
```
